# 🌐 Success Point Website

A modern and responsive educational website built using **HTML, CSS, and JavaScript**. This project is designed for coaching institutes or educational organizations to showcase their services, courses, achievements, and contact information.

---

## 📂 Project Structure

```
SuccessPoint/
│── index.html          # Main webpage
│── style.css           # Styling and responsive design
│── script.js           # JavaScript functionality
│── BACKGROUND.jpg      # Website background image
│── successspoint.jpg   # Logo / Hero image
│── README.md           # Project documentation
```

## ▶️ Installation & Usage

1. Download or clone this repository.
2. Keep all files in the same folder.
3. Open `index.html` in any modern web browser.

**OR**

Run using **VS Code** with the **Live Server** extension for the best experience.
